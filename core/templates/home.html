Automated Recruitment System

Intelligent Resume Screening & Skill Recommendation

SkillMatch AI introduces an automated approach that extracts, analyses, and evaluates resume data in a structured way using Natural Language Processing.

  • Supports two roles – Administrator (Recruiter) & Applicant (Job Seeker).
  • Recruiter posts jobs with required skills, qualifications, and job details.
  • Applicant creates an account, browses jobs and uploads resumes (PDF / DOCX).
  • System parses resume (name, contact, education, experience, skills) using NLP tools like Pyresparser.
  • Generates a Resume Score and a Skill Recommendation list to improve employability.
